## Getting Started

### Dependencies

macOS:
* brew (The macOS Homebrew)
* Python3
* nmap
* figlet

Linux Distribution: 
* Python3
* nmap
* figlet

### Installing

```
* macOS:
brew install git python3 python3-pip nmap figlet

* Linux:
sudo apt install git python3 python3-pip nmap figlet

* Automated Installaton:

git clone https://github.com/0n1cOn3/nmap-IP-Scanner.git
cd nmap-IP-Scanner
chmod +x scan.sh
```

### Executing program

```
./scan.sh
```
